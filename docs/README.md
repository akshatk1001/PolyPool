# Team Nodedemons: PolyPool

Polypool is a rideshare app intended for Cal Poly students to find rides back home for breaks and other occasions. The app allows users to create and join rides in a very easy and simple way, with transparency about cost and whether there are seats available. The app is locked to only cal poly emails, ensuring that only those who join are Cal Poly-associated.

This helps resolve the issue of rides being scattered across facebook and snapchat, making it difficult for students to find rides and for drivers to find riders. The app also allows users to split the cost of the ride, making it more affordable for everyone involved.
